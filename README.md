Esse é um projeto simples que fiz na faculdade como uma atividade. Um jogo da forca para o terminal em Python.

---Como funciona o jogo
O jogo escolhe aleatoriamente uma palavra de uma lista pré-definida.
A palavra é exibida inicialmente com traços (-) representando cada letra.
O jogador tem 5 tentativas para errar.

O jogo termina quando:
O jogador acerta a palavra completa ✅
Ou esgota todas as tentativas ❌
